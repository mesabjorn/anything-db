# Getting started

## Clone the repository

`git clone https://github.com/mesabjorn/anything-db.git`

## Install dependencies

```
cd .\anything-db\
python -m venv venv # optional
pip install -r .\requirements.txt
```

## Use the tool:

```
python main.py my_db_name.db
```
